# Incumbent API

The Incumbent API is a service that provides information about members of DPR and DPD. [API Pemilu](http://developer.pemiluapi.org/)

## How to setup

1. `$ bundle install`
2. Configure your database credentials in database.yml
2. `$ bundle exec rake db:setup`
3. `$ bundle exec rails server`

## Documentation API
See [Documentation API](http://docs.peraturanapi.apiary.io/) for information on defining and using Incumbent API.
